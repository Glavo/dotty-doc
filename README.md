# Dotty文档

Dotty 是一个为 Scala 尝试新语言概念以及编译器技术的平台。Dotty 的重点在于简化。我们删除无关紧要的语法（譬如没有 XML 字面量），并尝试将 Scala 的类型归结为更小的一些更基本的构建者。 DOT（依赖对象类型演算）研究了这些构建者的理论。

您通过本文档能够了解如何在机器上使用 Dotty 编译器，浏览代码，在您常用的 IDE 上配置 Dotty 开发环境。
